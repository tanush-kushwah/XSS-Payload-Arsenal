## 📥 Installation

### Option 1 — Clone with Git

```sh
# 1. Clone the repository
git clone https://github.com/tanush-kushwah/xss-payload-arsenal.git

# 2. Navigate into the folder
cd xss-payload-arsenal

# 3. Unzip the folder
unzip xss-extension-v2.zip
```

### Option 2 — Download ZIP

```sh
# Download and extract using curl
curl -L https://github.com/tanush-kushwah/xss-payload-arsenal/archive/refs/heads/main.zip -o xss-payload-arsenal.zip

# Unzip the file
unzip xss-payload-arsenal.zip

# Enter the folder
cd XSS-Payload-Arsenal-main

# Unzip the folder
unzip xss-extension-v2.zip

```

### Option 3 — Download Manually

1. Click the green **`Code`** button on this page
2. Select **`Download ZIP`**
3. Extract the ZIP to any folder on your computer

## 🌐 How to Import in Chrome

> Follow these steps exactly to load the extension into your browser.

**Step 1 — Open Chrome Extensions page**

```
chrome://extensions/
```

> Paste the above URL directly into your Chrome address bar and press **Enter**

---

**Step 2 — Enable Developer Mode**

```
Toggle the "Developer mode" switch → located in the TOP RIGHT corner of the page
```

---

**Step 3 — Load the Extension**

```
Click "Load unpacked"  →  Select the folder:  xss-extension-v2/
```

> ⚠️ Make sure you select the **root folder** (the one that contains `manifest.json`)

---

**Step 4 — Done!**

```
The ⚡ XSS Payload Arsenal icon will now appear in your Chrome toolbar.
Pin it for quick access: click the puzzle 🧩 icon → pin XSS Payload Arsenal
```
